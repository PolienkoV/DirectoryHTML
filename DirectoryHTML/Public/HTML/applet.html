Тег <applet>

Описание

Тег <applet> предназначен для вставки на страницу апплетов — небольших программ, написанных на языке Java. Этот тег является устаревшим, взамен необходимо использовать более гибкий тег <object>. Между открывающим и закрывающим тегом можно добавить текст, который будет отображаться в браузере, если он не поддерживает апплеты. В противном случае текст не выводится.

Синтаксис

<applet code="URL">Текст</applet>

align <applet align="bottom | left | middle | right | top"></applet> Поддерживается в браузерах: Internet Explorer, Chrome, Opera, Safari, Firefox Задает выравнивание апплета относительно близлежащих элементов и текста.
alt <applet code="URL" alt="текст"></applet> Ни один из браузеров не поддерживает альтернативный текст Атрибут alt устанавливает альтернативный текст для апплетов. Такой текст позволяет получить информацию о программе во время ее загрузки.
archive <applet code="URL" archive="URL файла с архивом"></applet> Данный атрибут не поддерживается ни одним из браузеров Позволяет указать архивный файл в формате ZIP или JAR (Java-архив), в котором располагается апплет.
code <applet code="URL"></applet> Имя файла с расширением .class. Если расширение не указано (например: code="clock"), то браузер добавит расширение автоматически. Указывает имя файла с Java-классом (оно имеет расширение .class). Это имя может комбинироваться со значением атрибута codebase, который задает путь к папке, где располагается файл.
codebase <applet code="URL" codebase="URL"></applet>

Текущее время:

Используется для установки базового адреса, относительно которого будет задаваться значение атрибута code. Если значение codebase не указано, то в качестве адреса принимается адрес текущего документа.
height <applet code="URL" height="значение"></applet>

Задаёт высоту апплета. Допускается использовать значения в пикселах или процентах. Если установлена процентная запись, то размеры апплета вычисляются относительно размера родительского элемента — контейнера, где находится тег <applet>.
hspace <applet code="URL" hspace="отступ по горизонтали"></applet> Задает горизонтальный отступ вокруг апплета.
vspace <applet code="URL" vspace="отступ по вертикали"></applet> Задает вертикальный отступ вокруг апплета
width <applet code="URL" width="отступ по ширине"></applet> Любое целое положительное число в пикселах или процентах.
Также для этого тега доступны глобальные атрибуты и события.
