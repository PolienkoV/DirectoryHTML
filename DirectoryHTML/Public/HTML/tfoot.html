Тег <tfoot>

Описание

Элемент <tfoot> предназначен для хранения одной или нескольких строк, которые представлены внизу таблицы. Хотя тег <tfoot> в исходном коде должен быть определен до тега <tfoot>, браузеры отображают его в самом низу таблицы. В пределах таблицы разрешается использовать только один элемент <tfoot>.

align <tfoot align="left | center | right | justify">...</tfoot>
Объединенная ячейка в TFOOT
Ячейка 1, расположенная в TBODY Ячейка 2, расположенная в TBODY
Задает выравнивание по горизонтали для всех ячеек в пределах <tfoot>. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для тега <td> или <th>.
bgcolor <tfoot bgcolor="цвет">...</tfoot>
Объединенная ячейка в TFOOT
Ячейка 1, расположенная в TBODY Ячейка 2, расположенная в TBODY
Устанавливает цвет фона ячеек, расположенных в контейнере <tfoot>.
char <table> <thead> .... </thead> <tfoot char="c" align="char"> ... </tfoot> <tbody> <tr> <td> ... </td> </tr> </tbody> </table> ... Атрибут char заставляет выравниваться содержимое ячеек внутри <tfoot> по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке или запятой. Работает только в том случае, если значение атрибута align установлено как char
charoff <table> <thead> .... </thead> <tfoot char="c" align="char"> ... </tfoot> <tbody> <tr> <td> ... </td> </tr> </tbody> </table> ... Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячеек влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево.
valign <tfoot valign="top | middle | bottom | baseline">...</tfoot>
Объединенная ячейка в TFOOT, выровнена по нижнему краю
Ячейка 1 Ячейка 2
Ячейка 3 Ячейка 4
Устанавливает вертикальное выравнивание содержимого ячеек в контейнере <tfoot>. По умолчанию контент ячейки располагается по ее вертикали в центре.
Также для этого тега доступны глобальные атрибуты и события.
