Тег <video>

Описание

Добавляет, воспроизводит и управляет настройками видеоролика на веб-странице. Путь к файлу задается через атрибут src или вложенный тег <source>.

autoplay <video autoplay="autoplay"> </video> При наличии этого атрибута видео начинает воспроизводиться автоматически после загрузки страницы. Атрибут autoplay отменяет действие preload.
controls <video controls="controls"> </video> Добавляет панель управления к видеоролику. Вид панели и ее содержимое зависит от браузера и может в себя включать: кнопка воспроизвести, пауза, перемотка, переход в полноэкранный режим; ползунок для изменения уровня громкости и др.
height <video height="высота"> </video> Атрибут height задает высоту области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в заданные рамки, но его пропорции при этом остаются прежними.
loop <video loop="loop"> </video> Зацикливает воспроизведение видео, оно повторяется каждый раз с начала после завершения
poster <video loop="loop"> </video> Указывает адрес картинки, которая будет отображаться, пока видео не доступно или не вопроизводится. Само изображение может быть в любом доступном формате: GIF, PNG, JPEG. Если атрибут poster не указан, браузер постарается отобразить первый кадр видео.
preload <video preload="none | metadata | auto"> </video> Используется для загрузки видео вместе с загрузкой веб-страницы. Этот атрибут игнорируется, если установлен autoplay
none
Не загружать видео.
metadata
Загрузить только служебную информацию (размеры видео, первый кадр, продолжительность и др.).
auto
Загрузить видео целиком при загрузке страницы
src <video src="URL"> </video> Указывает путь к воспроизводимому видеоролику. Для этой же цели можно использовать тег <source>.
width <video width="ширина"> </video> Атрибут width задает ширину области для воспроизведения видеоролика. Само видео меняет свои размеры в большую или меньшую сторону, чтобы вписаться в указанную ширину, но его пропорции при этом не искажаются
