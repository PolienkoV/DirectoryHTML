Тег <thead>

Описание

Элемент <thead> предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы. Допустимо использовать не более одного элемента <thead> в пределах одной таблицы, и он должен идти в исходном коде сразу после тега <table>.

align <thead align="left | center | right | justify | char">...</thead>
...
... ...
Задает выравнивание по горизонтали для всех ячеек в пределах <thead>. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для тега <td> или <th>.
bgcolor <thead bgcolor="цвет">...</thead>
... ...
... ...
Устанавливает цвет фона ячеек, расположенных в контейнере <thead>
char <table> <thead align="char" char="c"> <tr> <td> ... </td> </tr> </thead> <tfoot> ... </tfoot> <tbody> ... </tbody> </table>

Атрибут char позволяет выравнить содержимое ячеек внутри <thead> по заданному символу. Работает только в том случае, если значение атрибута align установлено как char.
charoff <table> <thead charoff="число"> <tr> <td> ... </td> </tr> </thead> <tfoot> ... </tfoot> <tbody> ... </tbody> </table>

Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячеек внутри <thead> влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево.
valign <thead valign="top | middle | bottom | baseline">...</thead>
...
... ...
... ...
Устанавливает вертикальное выравнивание содержимого ячеек в контейнере <tbody>. По умолчанию контент ячейки располагается по ее вертикали в центре.
Также для этого тега доступны глобальные атрибуты и события.
