Тег <optgroup>

Описание

Тег <optgroup> представляет собой контейнер, внутри которого располагаются теги <optgroup> объединенные в одну группу. Особенностью тега <optgroup> является то, что он не выделяется как обычный элемент списка, акцентируется с помощью жирного начертания, а все элементы, входящие в этот контейнер, смещаются вправо от своего исходного положения.

disabled <optgroup disabled>...</optgroup>

Блокирует доступ к группе списка, при этом группа отображается другим цветом (обычно серым) и недоступна для выбора.
label <optgroup label="текст">...</optgroup> ... Текст, который будет входить в список в виде заголовка группы. Это обычная текстовая строка без всяких кодов HTML
Также для этого тега доступны глобальные атрибуты и события.
