Тег <legend>

Описание

Тег <legend> применяется для создания заголовка группы элементов формы, которая определяется с помощью тега <fieldset>. Группа элементов обозначается в браузере с помощью рамки, а текст, который располагается внутри контейнера <fieldset>, встраивается в эту рамку.

accesskey <legend accesskey="c">Текст</legend>

Для перехода к блоку используется клавиша T.

Работа со временем

создание пунктуальности (никогда не будете никуда опаздывать);
излечение от пунктуальности (никогда никуда не будете торопиться);
изменение восприятия времени и часов.

Атрибут accesskey позволяет перейти к группе элементов формы с помощью некоторого сочетания клавиш с заданной в атрибуте буквой или цифрой. Браузеры при этом используют различные комбинации клавиш. Например, для accesskey="s" работают следующие сочетания.

  • Internet Explorer: Alt + S
  • Chrome: Alt + S
  • Opera: Shift + Esc, S
  • Safari: Alt + S
  • Firefox: Shift + Alt + S
